# Legacy Pilot Pipeline

This folder preserves the first Level 1 package generator used for the 20-word prototype.

It is not the active curriculum workflow. The active process uses the numbered scripts in `../scripts/` and the reviewed datasets in `../constructed_data/`. Keep this code only for reproducing or auditing the original pilot export.

The legacy scripts expect package and output files that are not part of the current workflow. Do not use their generated SQL or CSV files for production imports.
